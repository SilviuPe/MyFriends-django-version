# MyFriends-WebApp

## Features:
* ### Profile Customization
  You can customize your profile by changing your avatar, your name, password, email, etc ...
* ### Search for your friends profile:
  In the search input you can search for your friend and send them a friend request
* ### Animated notification
    When you have a new message from your friends an exclamation sign will popup close to their profile picture.
* ### File Transfering
  You can send files not only text messages, and the API it's made to filter the extenstions to prevent possible attackers to send unwanted files.
* ### Notifications
  In the notification section you can see who requested to be friend with you.

## Tools: 
* ### Backed:
    * Django Web Framework (A Python library)
* ### Frontend:
    * Simple HTML, CSS and JavaScript
